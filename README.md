# Snake Water Gun Game

A simple Python implementation of the Snake Water Gun game.

## About the Project

This is a beginner Python project based on the classic Snake Water Gun game, similar to Rock Paper Scissors.

## Features

* User vs Computer gameplay
* Random computer choices
* Score and winner determination
* Simple command-line interface

## Technologies Used

* Python

## How to Run

1. Clone the repository
2. Open the project folder
3. Run:

```python
python main.py
```
